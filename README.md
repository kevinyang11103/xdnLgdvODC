# 个性化音乐推荐系统 python1280

## 项目概述

本项目是基于Python开发的个性化音乐推荐系统，通过分析用户喜好和行为数据，为用户提供个性化的音乐推荐。

## 技术栈

- 后端框架：Django
- 前端框架：Vue.js
- 开发语言：Python（版本支持：3.7/3.8）
- 数据库：MySQL（版本：5.7或更高，理论支持5.X系列）
- 数据库工具：Navicat 11（推荐）

## 功能模块

- 用户管理：注册、登录、个人信息管理等
- 音乐管理：音乐上传、音乐信息编辑、音乐推荐等
- 数据分析：用户行为分析、音乐喜好分析等

## 系统角色

- 普通用户：使用系统进行音乐欣赏和互动
- 管理员：负责音乐管理和用户管理

## 运行环境

- Python环境（3.7/3.8版本）
- MySQL数据库环境（5.7或更高版本）
- Node.js环境（前端Vue.js依赖）

## 部署步骤

1. 配置Python环境（3.7/3.8版本）
2. 安装所需依赖库（Django、Vue.js等）
3. 配置MySQL数据库（5.7或更高版本）
4. 初始化数据库表结构
5. 运行后端服务
6. 运行前端服务

## 核心亮点

- 基于用户行为和喜好的个性化推荐算法，提供更精准的音乐推荐
- 前后端分离的设计，易于维护和扩展
- 支持多种开发工具（PyCharm、VS Code等）和数据库工具（Navicat 11等）
